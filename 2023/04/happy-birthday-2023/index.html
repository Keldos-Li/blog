祝我生日快乐~

发表于 2023-04-13 更新于 2023-05-10 阅读次数： Valine：

又一年了哇。祝我生日快乐。

或许生日的时机太不合适了。每年这个时候都是期中考，出图，论文，pre，考试，甚至体测……全都撞在一起了……今年还有hhy，天哪，本来就是每周都想去死了，而这周要完整的方案汇报。唉。太痛苦了。

这个学期，太难了，太难了。每周都是为了周一的交图而度过的。每周日，我早早来到专教，开始画图，啪，毫无节律的一天，晚上七八点吃个晚饭，十点跑个步，十一点去吃个宵夜，然后接着画，画到两三点，实在受不了了，回寝睡个觉，早上八点多起来接着画。大概率还是画不完的。午饭也可能来不及吃。翘了毛概课，然后在专教接着画到两点五十，然后急急忙忙打开学在浙大，把一坨图和ppt和pdf交上去。周二汇报完，就感觉一切都结束了。放空到周五，期间当然是还要赶别的课的作业的，然后，啪，又开始噩梦循环。

我不知道为什么会这样。为什么呢？

是够累了。我仿佛回到了大一那会儿。专教还在东教，我也还没自行车。每天晚上两三点，从顶楼走到地下车库，走一圈找不到共享单车，便慢慢沿着人行道从东教漫步回紫云，听着垃圾车在路上跑，路灯简单地发着黄光，我的影子长长的。这段路走得慢，大概要二十分钟。要是到四点，还能听见鸟儿在喳喳叫。回去的路上说不定还能碰上保洁阿姨骑着电驴往教学楼赶，人家上班我下班，不错。

现在倒是有自行车了，终于。戴上耳机，两点，整条车道就我一个人在路上晃晃悠悠地骑，放上 Kanye West 的 Run away，眼泪一下子就流下来了：为什么呢？为什么会过成现在这个样子呢？

我真的不理解。怎么就成了现在这个样子。一天，一天，一天，一周，一周，一周。我恨。

这样的生活真的，太没有意义了。我好像什么都没有学到。但我什么都要做。

说点开心的事吧。今天确实几乎啥也没干了。我想给自己放个假——当然，可能几天之后更来不及了。算了。通个宵没什么问题，要是不行，就通两个。

这学期，我至少做了些有意义的事情……或者说，能让我开心的事情。

我在给 https://github.com/GaiZhenbiao/ChuanhuChatGPT 打下手。截止我写这段话的时候，它已经有 8.7k star 了！

我知道我的主要读者都是不懂技术的，大概连github也不会上……所以我就不详细介绍了。大家明白这个项目为 ChatGPT/ChatGLM/LLaMA 等多种 LLM （大型语言模型）提供了一个轻快好用的Web图形界面 就行了。要是还是不明白，要不就理解为，它可以提供本地部署ChatGPT 的对话服务就行。

奇妙啊，写写CSS，JavaScript，管管issue、pr，写点文档，这些琐事，tmd我都感觉是一种美妙的消遣！debug 相比画图，都是多么美妙的体验啊！至少它真的能给我点成就感。而临浦镇呢？天哪，又有什么规划的意义呢？我真的没有动力给一个普普通通的小镇做假的规划。假题假做，反正一切都不会实现，做起来不过是苦逼画图工罢了。

哦，这部分我不该散布不好的情绪。

但今晚我确实很开心！啊，我可爱的朋友们。

烟囱已经第二年为我提供大会员服务了~（除了可能有点缺乏新意……但我确实很能用上这个hhh）

以及……感谢 川虎 和 昭昭！我收获了一大堆杭电计算机学院的纪念品！哈哈哈，不过我没有那么多钥匙能挂上那么多钥匙扣（）。也没有那么多鼠标能用上那么多鼠标垫（）。（我的桌子上全是源源头像，源源立牌，源源贴纸，甚至还有一个源源玩偶！我一直以为是猿猿，结果他俩告诉我是源源，好吧） 还有超大号的回车键！太奇怪了！它真的能用！

真好哇。

又12点多了。今天宜早睡。

各位晚安。明天会是美好的一天。（但我并不保证后天以及大后天以及周末和下周到学期末）
